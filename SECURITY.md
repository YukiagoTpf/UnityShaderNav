# Security Policy

UnityShaderNav is a local VS Code extension and language server. It parses local
shader files and Unity package metadata; it should not need network access at
runtime.

## Supported Versions

The project is in public-preview development. Security fixes are applied to the
current `main` branch until formal releases begin.

## Reporting a Vulnerability

Please do not open a public issue for a suspected vulnerability.

Send a private report to the repository owner with:

- A short description of the issue.
- Steps to reproduce.
- The affected file or command.
- Any possible impact on local files, VS Code execution, or package publishing.

The maintainer will acknowledge the report, investigate, and publish a fix or
advisory when appropriate.
